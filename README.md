### Программы 
Программы для автопилота написаны на языке **Python** с использованием библиотеки **kRPC**
- Программа для проверки готовности сервера к подключению
- Программа для посадки на Муну

### Цель проекта
Разработка и реализация автоматизированной системы управления космечским аппаратом в **KSP**, обеспечивающей оптимальный по расходу топлива и времени полёт от старта **с Кербина** (аналог Земли) до мягкой посадки **на поверхность Муны** (аналог Луны)

### Задачи проекта
1. Техническое исследование орбитальной механики полёта **Кербин-Муна**
2. Разработка математической модели полёта, включая:
    - оптимальные траектории выведения на орбиту
    - эффективные межорбитальные переходы
    - алгоритмы управляемой посадки в условиях отсутствия атмосферы
3. Проектирование и строительство ракеты с учётом требований и автоматизации
4. Создание программного обеспечения автопилота на языке **Python** с использование мода **kRPC**
5. Тестирование и автоматизация системы управления
6. Сравнительный анализ эффективности автоматизированного и ручного управления
